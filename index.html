Wnioski z prawa rozdzielności:

  • trzecie prawo (∃x : (p(x) ∧ q(x)) ⇒ [(∃x : p(x)) ∧ (∃x : q(x))]) mówi, że nie możemy skrócić zapisu koniunkcji kwantyfikatorów ∃,
  • czwarte prawo ([(∀x : p(x)) ∨ (∀x : q(x))] ⇒ ∀x : (p(x) ∨ q(x))) mówi, że nie możemy rozłożyć kwantyfikatora ∀ łączącego dwie formy zdaniowe implikacją,
  • wnioski te oczywiście wynikają z tego, że w prawach tych nie mamy równoważności.

Kontrprzykład (w nawiązaniu do trzeciego prawa):

Rozważamy dwie funkcje zdaniowe 'x = 2' oraz 'x = 3'. Dziedziną obu funkcji zdaniowych są liczby rzeczywiste. Wtedy zdania (∃x : x = 2) ∧ (∃x : x = 3) oraz ∃x : (x = 2 ∧ x = 3) są odpowiednio prawdziwe i fałszywe. Zatem fałszywa jest implikacja [(∃x : x = 2) ∧ (∃x : x = 3)] ⇒ ∃x : (x = 2 ∧ x = 3).

Kontrprzykład (w nawiązaniu do czwartego prawa):

Rozważmy dwie funkcje zdaniowe x > 0 oraz x ≤ 0. Dziedziną obu funkcji zdaniowych są liczby rzeczywiste. Wtedy prawdziwe jest zdanie ∀x : (x > 0 ∨ x ≤ 0). Natomiast alternatywa (∀x : x > 0) ∨ (∀x : x ≤ 0) jest fałszywa, gdyż obie składowe są fałszywe. Zatem fałszywa jest implikacja ∀x : (x > 0 ∨ x ≤ 0) ⇒ [(∀x : x > 0) ∨ (∀x : x ≤ 0)].
